# Visual Edges — Star Gate

Визуальный язык рёбер, используемый в визуализации Binary AST, Ownership-графа и Hyper-edges.

## Основной набор

| Символ | Название              | Смысл                              | Где применяется                              |
|--------|-----------------------|------------------------------------|----------------------------------------------|
| `—`    | Простое ребро         | Обычная связь                      | `EdgeKind::Control`, обычный `Data`          |
| `→`    | Направленное          | Направленный поток                 | Data-flow, направление передачи значения     |
| `△`    | Влияние               | Порождение / сильное влияние       | Создание нового значения                     |
| `◇`    | Устойчивая связь      | Стабильное двустороннее отношение  | Долгоживущие связи между объектами           |
| `□̸`   | Shared Borrow         | Разделяемое заимствование          | `OwnershipState::Borrowed`                   |
| `▲`    | Unique Borrow         | Уникальное (mutable) заимствование | `OwnershipState::BorrowedMut`                |
| `✕`    | Конфликт              | Конфликт владения                  | Одновременный shared + unique borrow         |
| `⬡`    | Гиперребро / Группа   | Атомарная группа, многоместная связь | `HyperMember`, `OwnershipGroup`, AtomicUpdate |

## Минимальный набор (5 символов)

```text
—   обычная связь
→   направление
△   влияние / порождение
✕   конфликт ownership
⬡   гиперребро / группа
```
